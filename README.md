# Coach Vilma — Landing

Sitio web de presentación para la Dra. Vilma G. Duschak (coaching ontológico profesional).

## Requisitos

- Node.js 18+
- npm

## Instalación y uso

```bash
npm install
npm run dev
```

Sitio en desarrollo: [http://localhost:3000](http://localhost:3000)

## Producción

```bash
npm run build
npm start
```

## Tecnologías

Next.js · React · Tailwind CSS · Framer Motion
